## 🎬 使用流程

### 第一次使用

1. **启动后端**
   ```bash
   cd backend
   python main.py
   ```

2. **启动前端**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

3. **访问应用**
   - 打开 http://localhost:5173
   - 查看Dashboard首页

4. **启动爬虫**
   - 在左侧面板点击"启动爬虫"
   - 等待完成
   - 所有页面数据自动更新

### 日常使用

1. **查看仪表板** - 快速了解数据概况
2. **分析趋势** - 查看数据变化趋势
3. **研究用户** - 了解用户贡献
4. **浏览内容** - 发现热门话题
5. **查询数据** - 搜索具体问题
