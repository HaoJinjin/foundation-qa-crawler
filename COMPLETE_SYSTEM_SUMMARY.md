## 📈 数据流

### 完整的用户操作流程

```
用户打开浏览器
    ↓
访问 http://localhost:5173
    ↓
App.vue 初始化
    ├─ Navbar 获取系统状态
    ├─ Sidebar 显示导航菜单
    └─ Dashboard 自动加载
    ↓
Dashboard.vue 页面
    ↓
useDataStore.fetchDashboard()
    ↓
apiClient.getDashboard()
    ↓
HTTP GET /api/v1/analysis/dashboard
    ↓
后端处理
├─ 检查缓存
├─ 读取JSON文件
└─ 返回数据
    ↓
前端接收JSON
    ↓
Store 更新状态
    ↓
Vue 组件自动重新渲染
    ↓
用户看到Dashboard数据 ✅
```

### 爬虫执行流程

```
用户点击"启动爬虫"
    ↓
store.startCrawler()
    ↓
HTTP POST /api/v1/crawler/start
    ↓
后端创建后台任务
    ├─ 生成任务ID
    ├─ 启动异步爬虫
    └─ 立即返回响应
    ↓
前端保存任务ID
    ↓
启动轮询
每2秒 GET /api/v1/crawler/task/{taskId}
    │
    ├─ 更新进度条
    ├─ 显示状态消息
    └─ 重复直到完成
    ↓
爬虫完成
    ↓
后端返回结果数据
    ↓
前端调用 refreshAllData()
    │
    ├─ fetchDashboard()
    ├─ fetchTrends()
    ├─ fetchUsers()
    ├─ fetchTags()
    └─ fetchQuestions()
    ↓
所有数据同步更新
    ↓
用户看到最新数据 ✅
```
